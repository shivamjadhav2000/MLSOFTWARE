About GOML

Machine Learning is the core subarea of Artificial Intelligence. It makes computers get into a self-learning mode without explicit programming. When fed new data, these computers learn, grow, change, and develop by themselves. Here our application, Machine Learning Desktop Application is a simple and user-friendly application that can be used to train Machine Learning models and lets you work quickly and more effectively. Whether you're new to programming or an experienced developer, it's easy to learn and use the application. User need to simply upload the dataset from the upload page and select the algorithm from the algorithm section and click on the build button to train the model. The software system will be more accurate at predicting outcomes without being explicitly programmed. This machine learning tool is relatively new to the market and is evolving quickly. Modal title

1Q) What is the specified format of the dataset?

1A) ML Desktop Application accepts datasets either in CSV(comma separated values) or XLSV(MS Excel) format.

2Q) Do user require any prior knowledge in programming to use the software?

2A) No, the user does not require any prior knowledge in either programming or ML Application implementation to use the software, but a little of knowledge is expected in choosing the algorithm for the appropriate problem statement.

3Q) Which software is being used in the backend? Is it tensor flow or Sci-kit learn?

3A) No third party module or framework is being used in the backend to implement ML Algorithms. We have written every algorithm from scratch using numpy module for vectorized array implementations. We created a module named new_brain using python that contains our vanilla algorithms.

4Q) Does the software require any dependencies to run?

4A) No, it does not require any dependencies to run the software. We package all the modules required for the software to run any environment. It is packaged into a executable file to run on any machine without any installation.

5Q) What if the user has zero knowledge about how to use the software?

5A) We built the software with a nice and user friendly dashboard the is easy to understand and use. All the options structured in their respective tabs. Also algorithms are placed in a nice hierarchy.

6Q) How much storage is required to use the software?

6A) The software utilizes less 500mb of the hard disk space to run.

7Q) How many algorithms available in the software?

7A) A total of 7 algorithms are available in the software namely:

==> Linear, Polynomial and Ridge in Regression Algorithms.

==> Support Vector Classifier, Logistic Regression and K Nearest Neighbor's in Classification Algorithms

==> K Means clustering in the Clustering Algorithms.
